# LITERATURE_REVIEW

Focus Area

Security Foundation

IEEE studies consistently highlight blockchain’s necessity for overcoming inherent security and data integrity vulnerabilities within urban IoT deployments. Centralized databases remain prime targets for cyberattacks.

Challenges

Implementation Hurdles

Major hurdles identified in recent literature include achieving adequate scalability for millions of devices, reducing network latency for real-time applications, and ensuring interoperability with legacy infrastructure.

Solutions

Emerging Approaches

Novel consensus algorithms (e.g., lightweight Proof of Stake), fault-tolerant protocols, and integrating Artificial Intelligence for proactive threat detection are leading research directions for 2025-2026.
